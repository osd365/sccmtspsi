# Task Sequence Deployment Orchestrator

**Advanced Task Sequence Orchestration for Microsoft Endpoint Configuration Manager**  

Task Sequence Deployment Orchestrator (also known as **SCCMTSPSI**) is a **free**, powerful tool designed to help organizations efficiently and consistently initiate Operating System Task Sequence deployments in Microsoft Endpoint Configuration Manager (formerly SCCM / MECM).

Built on industry best practices and informed by real-world expert insight, it delivers a **controlled, reliable deployment experience** across enterprise environments — reducing complexity, preventing data loss, improving build success rates, and enabling agile task sequence releases.

## Key Features

- **All-in-one modern interface** — user-friendly launcher for task sequences
- **Controlled Task Sequence Execution** with built-in content validation check
- **Automatic device staging** in MECM and Active Directory
- **BitLocker drive unlock** (MBAM, recovery key, file, or manual)
- **Smart detection** of available Task Sequences, Operating Systems, Applications & Office suites
- **Realm-based filtering** — show only approved / relevant items per environment or profile
- **Dynamic application & Office selection** with availability checks
- **Collection & AD group membership** — add computers automatically at build time
- **User data migration** — backup & restore (USB, network, hardlink) with data-loss prevention
- **Primary user assignment** & device affinity for better asset management
- **Disk formatting & labeling** — supports Legacy BIOS and UEFI modes
- **Device decommissioning** mode — clean wipe, AD/MECM removal
- **18+ extension attributes** — create multiple dynamic paths and variables
- **Robust automation & validation** — dependency checks, guardrails, notifications
- **Build reporting** — success/failure notifications + complete as-built documentation

## Why Use SCCM TS PSI?

- Replaces or enhances complex custom scripts
- Increases SOE (Standard Operating Environment) build success
- Enables safer, repeatable deployments
- Prevents common pitfalls like data loss or misconfigurations
- Free forever — no licensing costs
